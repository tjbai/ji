```
ji new (ji n):
    make new page

ji status (ji st) [-n N]:
    show status of past N days

ji touch (ji t):
    create new task

ji add (ji a):
    stage a task

ji restore, (j rs):
    unstage a task

ji push, (j p):
    mark all currently staged tasks as complete

ji edit, (j e), -n N=1:
    interactive editor

ji comment, jc:
    append comment to all staged tasks
```
