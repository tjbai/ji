```
ji n:
    create new page

ji st [-n N] [-v/-nv]:
    show working page

ji t CONTENT [-d D]:
    create a task

ji rm ID:
    remove a task

ji a ID:
    stage a task

ji rs ID:
    unstage a task

ji p:
    mark all currently staged tasks as complete

ji c CONTENT:
    append comment to all staged tasks

ji e:
    directly edit pages directory

ji b:
    generate html
```
